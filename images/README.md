# Smiski mascot images 🌱

Drop your own Smiski photos in this folder to replace the built-in drawn mascot.
The app looks for these three filenames (PNG, ideally square, transparent background best):

| filename | shown when |
|----------|-----------|
| `smiski-idle.png`  | welcome banner (calm, resting) |
| `smiski-happy.png` | celebration overlay when she switches trays |
| `smiski-tulip.png` | empty photo timeline (holding a tulip) |

If a file is missing, the app automatically falls back to the cute hand-drawn SVG —
so you can add just one, all three, or none.

## Getting images from Pinterest
Pinterest blocks automated downloads, so grab them by hand:
1. Open a pin you like from https://www.pinterest.com/delgiannap/smiski/
2. Right-click the image → **Save image as…** (or long-press → Save on mobile)
3. Rename it to one of the filenames above and drop it in this folder.

A quick free background remover (e.g. remove.bg) makes them look great over the pastel UI.
